The Micro-Expression Based Emotion Classification System (Relaxed vs Stressed) is a lightweight, browser-based prototype that detects subtle facial cues and classifies them into two emotional states by preprocessing uploaded or webcam-captured images into 32×32 grayscale thumbnails and applying pixel-level similarity through Mean Squared Error against a curated dataset; with its clean interface, multiple analysis modes, and privacy-friendly client-side execution, the system demonstrates how deep learning-inspired techniques can be implemented in real time without heavy computational requirements, making it an effective foundation for emotion-aware applications in education, healthcare, workplace monitoring, and research demonstrations.
